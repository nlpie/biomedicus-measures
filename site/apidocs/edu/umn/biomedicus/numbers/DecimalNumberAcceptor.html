edu.umn.biomedicus.numbers

Class DecimalNumberAcceptor

  • java.lang.Object
    • edu.umn.biomedicus.numbers.DecimalNumberAcceptor

  • public class DecimalNumberAcceptor
extends Object
    Finds decimal numbers in text, including fractions like 1 / 2 separated over separate tokens. Also detects hybrid ordinals, like "1st" "2nd" "3rd", etc.

    It is not safe to use an instance of this class from multiple threads at once, use multiple instances for concurrency.

    1.0.0
